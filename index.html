Dr. Seamus Clarke

I'm a postdoctoral researcher at the Theoretical Astrophysics group of the University of Cologne. I'm interested in star forming filaments, astrochemistry, synthetic observations and numerical methods.

Research

My research focuses on using hydrodynamic simulations to study how the diffuse gas of the Interstellar Medium (ISM) is able to condense into filaments and cores, leading to the formation of stars. I'm especially interested in the interplay between gravity and turbulence in dense gas filaments and the role this has on fragmentation. I am also interested in the chemical evolution of the ISM as gas moves from the diffuse gas into dense cores, and how this influences how these structures are seen when observed with dust and molecular line tracers.

Code development

During the course of my research I have developed and contributed to codes which are of use to the wider community:

  • I have developed a module, FragMent, which combines a number of fragmentation analysis techniques and model selection tools to characterise the fragmentation of filaments. This code is written as a python/C module and is open source.
  • I have developed a fully-automated spectral fitting routine, Behind The Spectrum (BTS), which allows for the fitting of entire data cubes of optically thin lines. This code is written as a python module and is open source.
  • I have contributed to the code, J-plots, which is a data analysis tool used to quantify the morphology of pixelated structures using their moments. It is written as a python module and is open source
  • I have contributed to the open source python package FilChaP, a tool for characterising filament properties such as their radial profiles.
